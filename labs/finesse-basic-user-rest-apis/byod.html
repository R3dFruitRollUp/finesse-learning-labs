To work on this lab on your own machine, install the following applications:

Install Cisco AnyConnect VPN Client Software

  • Open your browser
  • Go to https://developer.cisco.com/site/devnet/sandbox/anyconnect.
  • Follow the instructions to install Cisco AnyConnect.

Install Postman

  • Open your browser.
  • Go to https://www.getpostman.com.
  • Follow the instructions to install Postman.
