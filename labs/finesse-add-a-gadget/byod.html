To work on this lab on your own machine, you will need to install the following applications:

Install Cisco AnyConnect VPN Client Software

  • Go to https://developer.cisco.com/site/sandbox/anyconnect
  • Follow the instructions to install Cisco AnyConnect

Windows Users: Install a SSH Client - PuTTY

  • Go to http://www.putty.org
  • Follow the instructions to install PuTTY

Install a SFTP Client - FileZilla

  • Go to https://filezilla-project.org
  • Follow the instructions to install FileZilla
